# Cookie Clicker - OOP Game
Ayoub El Makrini & Younes El Hajri

## Beschrijving
Een Cookie Clicker game gebouwd met JavaScript OOP principes. Klik op cookies, koop autoclickers en upgrades, en bouw je cookie imperium!

## Features
- Cookie klikken met animaties
- 10 verschillende autoclickers
- 8 upgrades met multiplicatieve effecten
- 7 achievements met unlockable themes
- Save/Load systeem (LocalStorage)
- Dark/Light mode
- Responsive design
